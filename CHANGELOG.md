# Changelog

All notable changes to Eulard will be documented in this file.

## [0.1.0] - 2026-03-21

Initial release.

- AI-powered mermaid diagram editor with Claude integration
- Visual canvas (React Flow) with drag-and-drop nodes
- Code editor with live mermaid preview
- Diagram sharing and folder organization
- Google OAuth and email/password authentication
- API key authentication for programmatic access
- Admin dashboard for user and AI settings management
- Export to PNG, SVG, and mermaid code
- Chat sessions with persistent history
- Cloud Run deployment (dev + prod)
- GKE deployment infrastructure (k8s manifests + Terraform)
